🎯 QU'EST-CE QUE LE HACKING ÉTHIQUE ?

Un hacker éthique (ou pentesteur) est mandaté par une organisation pour tester ses défenses. La seule différence avec un attaquant malveillant : l'autorisation écrite.

Sans autorisation écrite → crime. Avec → profession reconnue et bien payée (70–130k€/an).

⚡ Ne pratique JAMAIS sur des systèmes que tu ne possèdes pas ou pour lesquels tu n'as pas d'autorisation explicite.

🎭 ARP SPOOFING & MITM

L'ARP Spoofing empoisonne le cache ARP des victimes pour te placer entre elles et la passerelle — tout le trafic passe par toi.

echo 1 > /proc/sys/net/ipv4/ip_forward▶ RUN
arpspoof -i eth0 -t VICTIME GATEWAY▶ RUN
bettercap -iface eth0▶ RUN
⚡ Bettercap est l'outil MITM moderne tout-en-un : ARP, DNS, HTTP, HTTPS stripping.
📡 WI-FI HACKING WPA2

Objectif : capturer le handshake WPA2 lors d'une (dé)authentification, puis le cracker offline.

airmon-ng start wlan0▶ RUN
airodump-ng wlan0mon▶ RUN
aireplay-ng --deauth 5 -a BSSID▶ RUN
hashcat -m 22000 capture.hc22000 rockyou.txt▶ RUN

🔑 IDENTIFIER & CRACKER LES HASH

Avant de cracker : identifier le type de hash. hashid ou hash-identifier font ça en secondes.

hashid 5f4dcc3b5aa765d61d8327deb882cf99▶ RUN
hashcat -m 0 (MD5) hash.txt rockyou.txt▶ RUN
hashcat -m 1000 (NTLM) hash.txt rockyou.txt▶ RUN
john --wordlist=rockyou.txt hash.txt▶ RUN
⚡ GPU Hashcat = 1 milliard de MD5/seconde. Un mot de passe de 8 chars → cracké en minutes.
🎫 JWT ATTACKS

Les JWT (JSON Web Tokens) sont partout en auth web. 3 attaques classiques :

1. alg:none — Change l'algorithme en "none", supprime la signature. Certains serveurs acceptent.

2. Brute force du secret — Si HS256, le secret peut être cracké.

hashcat -m 16500 token.jwt rockyou.txt▶ RUN

3. RS256→HS256 — Signe avec la clé publique comme secret HMAC.

⚡ Outils : jwt.io (décode), jwt_tool (automatise toutes les attaques), CyberChef.
🧮 PADDING ORACLE & CBC

Attaque sur les implémentations CBC qui révèlent si le padding est valide. Permet de déchiffrer sans clé.

Principe : chaque bloc CBC dépend du précédent. En manipulant les octets et observant les erreurs de padding, on reconstruit le plaintext octet par octet.

padbuster target CIPHER BLOCKSIZE▶ RUN
⚡ Ressource : CryptoHack.org — challenges progressifs sur tous les algos. Indispensable pour les CTF crypto.
